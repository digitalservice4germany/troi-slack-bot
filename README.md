# troi-slack-bot
Slack bot to help you fill out Troi.

Follow [this](https://slack.dev/bolt-js/tutorial/getting-started) tutorial for the required Setup in Slack.

## Setup

```js
npm i
```

## Run

```js
export SLACK_BOT_TOKEN=...
export SLACK_SIGNING_SECRET=...
export APP_TOKEN=...
node app.js
```
